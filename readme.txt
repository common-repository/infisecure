== Description ==

InfiSecure is a robust web security platform to detect and block online threats. The service bundles all aspects of website security, including Bot Protection, Click Fraud Protection and Fake Traffic Prevention.

InfiSecure is built with a belief that businesses should be able to extract the benefits of the Internet without the fear of risk or compromise.

Major features in InfiSecure include:

* Known Violators Testing.
* IP Behavior Analysis.
* User Behavior Analysis.
* Device Fingerprinting.
* Centralized Intelligence.
* Machine Learning Algorithms.

PS: You'll need an [Infisecure.com API key](https://www.infisecure.com/pricing.html) to use it.  .

== Installation ==

Upload the Infisecure plugin to your blog, Activate it, then enter your credentials which you can get from https://www.infisecure.com.

1, 2, 3: You're done!
